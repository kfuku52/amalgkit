## Legacy page

`amalgkit csca` is not exposed by the current CLI. This page is kept only as historical reference for older workflows.

## What to use now

For the current workflow, use:

- [`amalgkit wsfilter`](https://github.com/kfuku52/amalgkit/wiki/amalgkit-wsfilter) for within-species outlier filtering
- [`amalgkit csfilter`](https://github.com/kfuku52/amalgkit/wiki/amalgkit-csfilter) for cross-species outlier filtering
- [`amalgkit finalize`](https://github.com/kfuku52/amalgkit/wiki/amalgkit-finalize) for final table export and batch correction

`csfilter` now provides the main cross-species QC figures directly, including PCA, overview, heatmap, and outlier plots.
